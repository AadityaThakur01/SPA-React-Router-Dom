## ‚ú® Features

- **Client-Side Routing:** Navigate between pages (Home, About, Contact) without refreshing the browser.
- **Layout Architecture:** Uses a master `Layout` component with an `<Outlet />` to keep the Header and Footer consistent across all pages.
- **Active Link Styling:** The navigation bar highlights the current page automatically using `NavLink`.

- **Responsive Design:** Fully responsive UI built with Tailwind CSS.
